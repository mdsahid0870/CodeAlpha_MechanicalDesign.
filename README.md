# CodeAlpha_MechanicalDesign.
Task 1: 3D CAD Model Creation — Crankshaft
Objective
The objective of this task is to design a detailed 3D CAD model of a mechanical component using Autodesk Fusion 360, applying proper engineering dimensions and material selection.

Overview
This task is part of the Mechanical Design & Simulation Internship at CodeAlpha. A Crankshaft was selected as the mechanical component — a critical part of an automotive engine that converts the reciprocating motion of pistons into rotational motion.

Steps Performed
Component Selection: Selected a single-cylinder crankshaft as the design component.
3D Modeling: Modeled all major features including main journals, crank pins, crank arms, counterweights, and flanges using Autodesk Fusion 360.
Dimensions: Applied proper engineering dimensions based on standard single-cylinder engine specifications.
Material Selection: Assigned Alloy Steel (AISI 4340) for its high strength, fatigue resistance, and toughness — properties essential for rotating engine components.
Rendering: Generated a photo-realistic rendered image of the final model.
Technical Drawing: Created a 2D engineering drawing with all critical dimensions, tolerances, and section views.
How to Open
Open Crankshaft.f3d in Autodesk Fusion 360 to view the complete parametric 3D model with full feature tree.
